This climada  module contains the European winter storm hazard event sets as used in the following publication:
Cornelia Schwierz, Pamela Koellner-Heck, Evelyn Zenklusen Mutter, David N. Bresch, Pier-Luigi Vidale, Martin Wild, Christoph Schaer, 2010:Modelling European winter wind storm losses in current and future climate.

The key results have also been published in the brochure 'The effect of climate change: Storm damage in Europe on the rise'.

The entity is a dummy entity to test the hazard sets, it contains the damage function as used in the paper. Please consider using GDP_entity to create meaningful European country assets etc.
